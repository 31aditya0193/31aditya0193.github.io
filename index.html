Soteria

Keeping a record of your vital stats helps us in a long run. I decide to make an app which in keep and eye on vitals of your self or your dear ones, even if you are in the another corner of the world. It also keep a tab if some vital reading is going into dangerous zone. I used JSONSerialization after getting data from server via URLSession. The data is sent from HealthKit to the server so that our closed ones can know that we're doing well.

Used stack:

  • Swift
  • SwiftUI
  • URLSession
  • RESTful API




Github: Soteria

ChartKit Framework

This project started as an experiment using SPM and SwiftUI Shaped. The idea was simple: create a library which lets user to forget about plotting worries and draw them by supplying only data. For the distribution I have chosen SPM but thinking to expand it to cocoaPods in later stage. Plotting and charting is handled using swiftUI Shapes.

Used stack:

  • Swift
  • SPM
